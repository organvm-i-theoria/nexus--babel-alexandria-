# Security Policy

## Supported Versions

Theoria Linguae Machina (RLOS) is currently in **Phase 0 (Foundation & Infrastructure)**. There are no stable release versions yet. Development branches are considered unstable.

| Version | Supported          |
| ------- | ------------------ |
| Phase 0 | :warning: Unstable |

## Reporting a Vulnerability

We take the security of RLOS seriously. If you discover a security vulnerability, please report it privately.

**DO NOT** report security vulnerabilities through public GitHub issues.

### How to Report

Please email `security@example.com` with:
- A description of the vulnerability.
- Steps to reproduce.
- Potential impact.

We will acknowledge your report within 48 hours.

### Scope

This project involves complex linguistic and AI components. We are particularly interested in:
- Model safety and alignment issues.
- Data privacy concerns in the corpus processing pipeline.
- Infrastructure vulnerabilities (API, auth, etc.).

## Security Measures

- We use automated dependency scanning (planned).
- We enforce code reviews for all changes.
- We utilize static analysis tools (mypy, bandit, etc.).
