# Rhetorical-Linguistic Operating System (RLOS)

## Overview
RLOS is a comprehensive system for rhetorical and linguistic analysis, currently in Phase 0 (Foundation & Infrastructure).

## Development
This project uses Poetry for dependency management and Ruff for linting/formatting.

### Setup
```bash
poetry install
```

### Performance
We prioritize performance. Run linting before committing:
```bash
ruff check .
```
