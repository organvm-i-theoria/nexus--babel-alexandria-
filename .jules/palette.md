## 2024-05-23 - Documentation Navigation as UX
**Learning:** In the absence of a UI, the file structure and naming conventions *are* the user interface. Unwieldy filenames and missing READMEs create immediate friction for developers (the users).
**Action:** Always ensure a clear `README.md` entry point and normalize filenames to be standard (`DESIGN.md` vs `# ...`) to support accessibility and ease of use.
